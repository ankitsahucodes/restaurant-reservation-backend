# Restaurant Reservation System - Backend

Backend API built with Node.js, Express.js, MongoDB, and JWT Authentication.

## Setup Instructions

### Clone Repository

```bash
git clone https://github.com/ankitsahucodes/restaurant-reservation-backend.git
cd restaurant-reservation-backend
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file.

```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
FRONTEND_URL=http://localhost:5173
```

### Run the Server

```bash
npm start
```
